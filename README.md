**TODOS** is a mini app for planning with stylish design.  
And what you can do with it:
​
* add tasks
* mark tasks as completed
* view active, completed and all tasks in separate lists
* delete completed tasks
​
## Used
​
- Vite-template-redux
- Material UI 
- Type Script
- Redux Toolkit
- React Testing Library
- Сodeclimate
